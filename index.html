📖 项目概览

SharpConfig 是一个面向 .NET 平台的 CFG/INI 配置文件操作类库，由 Cem Dervis 创建并维护。 你可以使用它以文本或二进制格式读取、修改和保存配置文件及流。该库基于 .NET Standard 2.0，可在 Windows、Linux、macOS 及 Docker 中无缝运行。

💡 提示：安装后在代码文件顶部添加 using SharpConfig; 即可使用。

🚀 快速入门

假设配置文件 sample.cfg 内容如下：

ini # 注释 [General] SomeString = Hello World! SomeInteger = 10 SomeFloat = 20.05 SomeBoolean = true SomeArray = { 1, 2, 3 } Day = Monday [Person] Name = Peter Age = 50

读取配置

csharp var config = Configuration.LoadFromFile("sample.cfg"); var section = config["General"]; string someString = section["SomeString"].StringValue; int someInteger = section["SomeInteger"].IntValue; float someFloat = section["SomeFloat"].FloatValue; bool someBool = section["SomeBoolean"].BoolValue; int[] someIntArray = section["SomeArray"].IntValueArray; DayOfWeek day = section["Day"].GetValue<DayOfWeek>();

内存创建配置

csharp var myConfig = new Configuration(); myConfig["Video"]["Width"].IntValue = 1920; myConfig["Video"]["Height"].IntValue = 1080; myConfig["Video"]["Formats"].StringValueArray = new[] { "RGB32", "RGBA32" }; myConfig["Video"]["Fullscreen"].BoolValue = true;

📚 API 参考

Configuration

方法说明
LoadFromFile / LoadFromStream加载文本配置
LoadFromBinaryFile / LoadFromBinaryStream加载二进制配置
SaveToFile / SaveToBinaryFile保存配置
this[string name]获取或创建 Section

Section & Setting

Section 通过索引器获取 Setting；Setting 提供大量强类型属性如 IntValue、StringValueArray 以及泛型方法 GetValue<T>()。

⚙️ 全局选项

选项说明
PreferredCommentChar默认 '#'
ValidCommentChars默认 { '#', ';' }
ArrayElementSeparator默认 ','
IgnoreInlineComments是否忽略行内注释
⚠️ 中文编码：加载/保存时建议指定 UTF-8 或 GB2312 编码。
