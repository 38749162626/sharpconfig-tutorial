📖 读取配置
var config = Configuration.LoadFromFile("sample.cfg"); var section = config["General"]; string someString = section["SomeString"].StringValue; int someInteger = section["SomeInteger"].IntValue; float someFloat = section["SomeFloat"].FloatValue; bool someBool = section["SomeBoolean"].BoolValue; int[] someIntArray = section["SomeArray"].IntValueArray; DayOfWeek day = section["Day"].GetValue<DayOfWeek>();

安装

dotnet add package sharpconfig
Install-Package sharpconfig

示例

一个示例配置文件 (sample.cfg)：

[General] # 注释 SomeString = Hello World! SomeInteger = 10 # 行内注释 SomeFloat = 20.05 SomeBoolean = true SomeArray = { 1, 2, 3 } Day = Monday [Person] Name = Peter Age = 50

读取这些值的 C# 代码如下：

var config = Configuration.LoadFromFile("sample.cfg"); var section = config["General"]; string someString = section["SomeString"].StringValue; int someInteger = section["SomeInteger"].IntValue; float someFloat = section["SomeFloat"].FloatValue; bool someBool = section["SomeBoolean"].BoolValue; int[] someIntArray = section["SomeArray"].IntValueArray; string[] someStringArray = section["SomeArray"].StringValueArray; DayOfWeek day = section["Day"].GetValue<DayOfWeek>(); // 用户自定义对象可以与 Section 相互转换 var person = config["Person"].ToObject<Person>();

加载

var cfg1 = Configuration.LoadFromFile("myConfig.cfg"); // 从文本文件加载 var cfg2 = Configuration.LoadFromStream(myStream); // 从文本流加载 var cfg3 = Configuration.LoadFromString(myString); // 从字符串加载 var cfg4 = Configuration.LoadFromBinaryFile("myConfig.cfg"); // 从二进制文件加载 var cfg5 = Configuration.LoadFromBinaryStream(myStream); // 从二进制流加载

遍历

foreach (var section in myConfig) { foreach (var setting in section) { // ... } }

在内存中创建配置

var myConfig = new Configuration(); myConfig["Video"]["Width"].IntValue = 1920; myConfig["Video"]["Height"].IntValue = 1080; myConfig["Video"]["Formats"].StringValueArray = new[] { "RGB32", "RGBA32" }; int width = myConfig["Video"]["Width"].IntValue; int height = myConfig["Video"]["Height"].IntValue; string[] formats = myConfig["Video"]["Formats"].StringValueArray;

保存

myConfig.SaveToFile("myConfig.cfg"); // 保存为文本文件 myConfig.SaveToStream(myStream); // 保存为文本流 myConfig.SaveToBinaryFile("myConfig.cfg"); // 保存为二进制文件 myConfig.SaveToBinaryStream(myStream); // 保存为二进制流

全局选项

通过修改 Configuration 类的静态属性可以自定义 SharpConfig 的行为。

选项类型描述
CultureInfoCultureInfo用于值转换的区域信息，默认为 InvariantCulture。
ValidCommentCharschar[]有效的注释分隔字符数组，默认：{ '#', ';' }
PreferredCommentCharchar保存时的首选注释字符，默认：'#'。
ArrayElementSeparatorchar数组元素分隔符，默认：','。
IgnoreInlineCommentsbool是否在解析时忽略行内注释。
IgnorePreCommentsbool是否在解析时忽略前置注释。
SpaceBetweenEqualsbool保存时是否在等号两侧添加空格。
OutputRawStringLiteralsbool字符串值是否不带引号写出。

忽略属性、字段与类型

将 [SharpConfig.Ignore] 特性应用于属性、字段或整个类，即可在映射时排除它们。

class SomeClass { public string Name { get; set; } public int Age { get; set; } [SharpConfig.Ignore] public int SomeInt { get; set; } } // 也可以直接忽略整个类型： [SharpConfig.Ignore] class ThisTypeShouldAlwaysBeIgnored { }

添加自定义对象转换器

步骤1：通过继承 TypeStringConverter<T> 创建自定义转换器。

public class PersonStringConverter : TypeStringConverter<Person> { public override string ConvertToString(object value) { var person = (Person)value; return string.Format("[{0};{1}]", person.Name, person.Age); } public override object ConvertFromString(string value, Type hint) { var split = value.Trim('[', ']').Split(';'); return new Person { Name = split[0], Age = int.Parse(split[1]) }; } }

步骤2：注册该转换器。

Configuration.RegisterTypeStringConverter(new PersonStringConverter());
常见问题

有疑问？这里都有答案

配置文件中文乱码怎么办？
加载时指定编码即可：Configuration.LoadFromFile("app.ini", Encoding.UTF8)。
如何忽略所有注释？
设置 Configuration.IgnorePreComments = true; 和 IgnoreInlineComments = true;。
是否支持嵌套节？
原生不支持，但可通过命名约定（如 [Parent.Child]）模拟层级结构。
索引器找不到键会报错吗？
不会。SharpConfig 使用“获取或创建”语义，访问不存在的键会自动创建，非常适合动态构建配置。
