📖 项目概览

SharpConfig 是 Cem Dervis 开发的 .NET 配置管理库，旨在用最少的代码实现 CFG/INI 文件的读取、修改和保存。它基于 .NET Standard 2.0，无任何第三方依赖，可在 Windows、Linux、macOS 上完美运行。

🚀 快速入门

假设我们有 app.cfg 文件：

[General] SomeString = Hello World! SomeInteger = 10 SomeFloat = 20.05 SomeBoolean = true SomeArray = { 1, 2, 3 } Day = Monday

读取配置：

var config = Configuration.LoadFromFile("app.cfg"); var section = config["General"]; string str = section["SomeString"].StringValue; int num = section["SomeInteger"].IntValue; float flt = section["SomeFloat"].FloatValue; bool flag = section["SomeBoolean"].BoolValue; int[] arr = section["SomeArray"].IntValueArray; DayOfWeek day = section["Day"].GetValue<DayOfWeek>();
💡 索引器是“获取或创建”语义，访问不存在的键会自动创建，非常适合动态构建配置。

❓ 常见问题

配置文件中文乱码怎么办？
保存时指定编码：Configuration.LoadFromFile("app.ini", Encoding.UTF8)。
如何忽略所有注释？
设置 Configuration.IgnorePreComments = true; 和 IgnoreInlineComments = true;。
是否支持嵌套节？
不支持。但可以通过命名约定（如 [Parent.Child]）模拟层级。
